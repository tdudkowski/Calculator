This is my first calculator written in JavaScript. It works, but is still buggy, especially if many dots are in operation.

  • Dividing by zero is illegal, and not supported
  • All division results are rounded to fifth position after the dot
  • Only the last one operator is counted, except situation when the last one is minus, so...
  • If the last one operator is minus, then next number is negative, and the previous one operator is counted, except situation when the minus is the only one, so...
  • If the minus is the only one, this is just the minus.
  • Dot is a decimal. All dots from the beginning of a number are removed, so .1 is just a 1 not 0.1, as one could expect. Only the first dot in a number is sounting, the second one and all signs after it are deleted.
